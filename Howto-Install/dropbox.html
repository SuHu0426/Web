Dropbox

安裝步驟

  1. 安裝相依套件.

    2. $ sudo aptitude install libnautilus-extension-dev checkinstall
$ sudo aptitude install python-docutils

  2. 下載安裝 Dropbox.

    3. $ wget http://www.dropbox.com/download?dl=packages/nautilus-dropbox-0.6.9.tar.bz2
$ tar xjvf nautilus-dropbox-*
$ cd nautilus-dropbox-*
$ ./configure
$ make
$ sudo checkinstall make install
