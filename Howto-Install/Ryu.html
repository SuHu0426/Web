Install

pip install

sudo aptitude install python-pip libxml2-dev libxslt1-dev
sudo pip install ryu

git install

sudo aptitude install python-dev libxml2-dev libxslt1-dev
git clone git://github.com/osrg/ryu.git
cd ryu; sudo python ./setup.py install

OpenFlow Tutorial

You will need the following installed in your environment that runs Ryu

  • python-eventlet
  • python-routes
  • python-webob
  • python-paramiko 
sudo aptitude install python-eventlet python-routes python-webob python-paramiko

running Ryu

PYTHONPATH=. ./bin/ryu-manager ryu/app/simple_switch.py

This example runs simple_siwtch ryu application. If you have your own ryu application, replace ryu/app/simple_switch.py with yours. By adding --verbose option, you can observe more logs from Ryu.
