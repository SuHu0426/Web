Debian Apache Maven 3 Installation Guide

Reference Apache Maven

Process

  1. Install Oracle Java
  2. Download Maven Binary
  3. Install Maven 3 
    $ wget http://apache.cdpa.nsysu.edu.tw/maven/maven-3/3.2.3/binaries/apache-maven-3.2.3-bin.tar.gz
$ tar zxvf apache-maven-3.2.3-bin.tar.gz
$ sudo cp -r bin boot conf lib /usr/local/
$ java -version
