中興資工 GMail 帳號使用說明

  1. 第一次登入 GMail
  2. Outlook Express 設定
  3. 同步舊信箱

第一次登入 GMail

回到目錄

  1. 使用瀏覽器登入 Gmail 信箱 http://mail.google.com/a/cs.nchu.edu.tw

  2. 登入帳號，經由一般 Gmail 網址登入需輸入完整名稱 ex: account@cs.nchu.edu.tw
    以專用入口http://mail.google.com/a/cs.nchu.edu.tw，則只需輸入帳號名即可（不需包含 domain @cs.nchu.edu.tw）
    預設密碼為 cs2013mail

  3. 輸入左下方之認證碼後點選 "我接受，請轉到我的帳戶"

  4. 第一次登入時需設定您的新密碼，目前密碼為 cs2013mail

  5. 登入成功

Outlook Express 6 (Windows XP) 設定說明

回到目錄

  1. 點選 "工具"，接著點選 "帳戶"

  2. 選擇到 "郵件" 子項目下，之後點選 "新增 -> 郵件"

  3. 輸入顯示名稱

  4. 新增帳戶，輸入完整 mail 名稱

  5. 設定 pop3 與 smtp。
    POP3 伺服器為 pop.gmail.com
    SMTP 伺服器為 smtp.gmail.com

  6. 帳戶名稱請輸入完整帳號 ex: account@cs.nchu.edu.tw
    密碼則是第一次登入後所設定之密碼，請務必完成網頁第一次登入手續再設定 POP3

  7. 完成帳號設定後，回到剛剛建立的郵件帳戶，點選 "內容"

  8. 移動到 "伺服器" 項目下
    確定帳戶名稱有包含 @cs.nchu.edu.tw
    勾選外寄郵件伺服器項目下之 "我的伺服器需要驗證"

  9. 移動到 "進階" 項目下。
    外寄郵件 - SMTP port 為 465
    內送郵件 - POP3 port 為 995
    兩者皆需勾選 "這個伺服器需要安全連線 - SSL"

同步舊信箱

回到目錄

  1. 點選 Gmail 右上方 "齒輪圖示" 後，點選 "設定"

  2. 接著切換到 "帳戶" 子項目下

  3. 點選 "新增您的 POP3 郵件帳戶"
  4. 在電子郵件地址欄位中輸入您的原先的帳戶名，但是需將 Domain 換成 oldmail.cs.nchu.edu.tw
    ex: demo@oldmail.cs.nchu.edu.tw

  5. 接下來依序填入：使用者名稱、密碼、POP 伺服器（pop.cs.nchu.edu.tw）、通訊埠（110）。


  6. 完成新增郵件帳戶後，選擇 "否"，我們已不再提供郵件發送服務。（以 Gmail 發送即可。）

  7. 最後確認信件已經備份完畢之後，點選 "刪除"，將帳戶刪除。（新的信件將直接送往新信箱，因此沒有必要繼續收舊信箱的信。）
