  2. Port (通訊埠)：

    3. 每一種電腦通訊協定都有預設的通訊埠。 HTTP port 通常使用 80，而 SSH port 通常使用 22。 本次開放系統作業，資工系提供 100 台虛擬機器。 如果全部使用實體 IP，除了安全性的問題、也相當的浪費資源。 所以利用一組實體 IP (網址)，開放額外的通訊埠，轉址到內部的區域網路，解決實體 IP 不足的問題。

  3. 連線方式：

    4. 透過 MobaXterm 或是 PuTTY SSH 連線軟體.。 使用 MobaXterm 以及 port 10022 為例： 輸入指令 ssh -X cloud@cloud.cs.nchu.edu.tw -p 10022。 第一次連線時，會建立認證金鑰並詢問 yes or no，再輸入密碼即可成功登入。

    傳輸檔案時，使用 WinSCP。 在 Host name 的地方輸入網址(IP)、填入自己的 port。 並且在 User name 的地方輸入帳號 cloud，即可點選 login 成功登入。

  4. 網頁伺服器安裝：
