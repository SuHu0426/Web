上這堂開放系統,是我第一次接觸到MS和OSX以外的作業系統,正巧研究所也是接觸coding
所以也是用Linux,但是差異在於我們是用Ubuntu
因為我有接觸一些WebCrawler和Package的東西,所以感覺的到Linux系統的強大之處
舉例來說:像我要安裝Python的話,Linux已經是內建Python3.3,還有再安裝Python_Scapy這個Lib的時候
Window必須安裝winpcap,scapy,pywin,pypcap,libdnet,pyreadline..一堆安裝檔還要版本全部都相符才可以
相較之下Linux只要輸入

    $ cd /tmp
    $ wget scapy.net
    $ unzip scapy-latest.zip
    $ cd scapy-2.*
    $ sudo python setup.py install
    $ sudo apt-get install tcpdump graphviz imagemagick python-gnuplot python-crypto python-pyx

就安裝好了,實在省時需多,而且windows用起來會有很多問題,linux就不會有這些問題
這堂課主要學到一些Linux基本的指令和一些shell發展的歷史和概念
對於沒有學過資料結構的我們來講是很實用的東西
還有學到如何用虛擬機來安裝一個Linux系統,畢竟不是每個人都有兩台電腦,所以有虛擬機的話真的很方便
否則的話整台電腦都裝Linux肯定吃不消,畢竟已經使用Windows十幾年有了,玩遊戲或者是做ppt都是Windows略勝一籌

然後上機實習課其實都不太清楚自己在幹嘛,大概只有分割硬碟這部分才知道自己在做些什麼
不然大部分的時間都在複製貼上,而且時間很趕(也有可能是同時一堆電腦連線所以網路處理非常慢的關係)
所以覺得好像沒有學到什麼東西的感覺,而且我查wiki是說debian是適合當作伺服器的版本
因為他有絕佳的穩定性,但是其實一般在當PC使用的,據我所知應該都是用Ubuntu居多
我自己用過的感覺是Ubuntu擁有比較好的GUI介面和軟體相容性,相較之下debian很多軟體都是舊版的
但也有可能是因為debian為求穩定度所以發放憑證的門檻較高,所以也無可厚非就是了
